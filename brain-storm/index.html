Ваша задача - запомнить показанное число и воспроизвести его с помощью клавиатуры. Ограниченный по длительности показ чисел приучает Вас к минимальному времени фиксации при чтении. Старайтесь фотографировать образ числа максимально большими фрагментами, без внимательного фокусирования на отдельных цифрах и по возможности их про себя не прогаваривать. Это упражнение служит для расширения поля зрения по горизонтальной периферической области. Это очень полезное упражнение не только для быстрого чтения, но и для других видов деятельности человека.
